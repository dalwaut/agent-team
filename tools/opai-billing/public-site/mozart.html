What is Mozart?

Mozart is an agentic orchestration system where a central conductor (the Orchestrator) decomposes complex goals into subtasks, assigns them to specialized agent squads, gathers real-time feedback from an "Audience" layer (evaluators, logs, critics), and closes the loop through a Reviewer that generates improvements — creating continuous evolution with every run.

Multi-Agent Parallel Execution

Decompose and run subtasks simultaneously across squads

Real-Time Audience Feedback

Bots, logs, and optional human signals evaluate progress live

Self-Improving Reviewer Loop

Automatic critique + instruction refinement for next iterations

Persistent Learning Library

Every run feeds back into long-term knowledge & patterns

Human-in-the-Loop Ready

Easy intervention points without breaking flow

Scalable & Composable

Build simple scripts or enterprise-grade agent swarms
