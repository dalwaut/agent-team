What is Mozart?

Mozart is a powerful agentic system: a central Orchestrator decomposes goals into subtasks, dispatches them to parallel agent squads, collects live feedback from an Audience layer (evaluators + logs + optional human input), and closes the loop via a Reviewer that auto-generates refined instructions — fueling continuous improvement with every execution.

Live Audience Evaluation

Self-Improving Feedback Loop

Reviewer auto-refines prompts and strategies for future runs

Persistent Knowledge Library

Every cycle enriches long-term patterns and reusable artifacts

Human-in-the-Loop Integration

Seamless intervention points at any stage

From quick scripts to massive agent swarms
