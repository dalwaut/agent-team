## Enhanced Knowledge: Data Storytelling [business-analytics/data-storytelling]
- Narrative arc: Hook (surprising insight) -> Context (baseline) -> Rising Action (data points) -> Climax (key insight) -> Resolution (recommendations) -> Call to Action
- Three pillars: Data (numbers, trends, comparisons), Narrative (context, causation, implications), Visuals (charts, highlights)
- Story frameworks: Problem-Solution (quantify pain, show pattern, propose fix with ROI), Trend (before/after transformation), Comparison (weighted scoring matrix)
- Headlines formula: [Specific Number] + [Business Impact] + [Actionable Context] -- e.g., "We're Losing $2.4M to Preventable Churn"
- Progressive reveal technique: start simple (one chart), add layers (growth rate overlay, segment breakdown, market share, opportunity)
- Contrast and compare: Before/After tables, side-by-side metrics, weighted scoring matrices for decisions
- Writing rules: start with "so what" (lead with insight), rule of three, show don't tell, make it personal to audience goals
- Handle uncertainty honestly: state confidence levels, present ranges, acknowledge sample size and correlation vs causation
- One-page dashboard story: Headline -> Key Metrics at a Glance -> What's Working -> What Needs Attention -> Root Cause -> Recommendation -> Next Focus
- Avoid: data dumps, burying the insight, jargon mismatched to audience, showing methodology before context, numbers without narrative
