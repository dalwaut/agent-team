PHASE 1: SCAN

- notes/Access/ — credentials, API keys, tokens, passwords

- notes/Coolify/ — Coolify/Docker infrastructure notes
- notes/Posts/ — content drafts, blog posts, social media
- notes/Review/ — INBOX for unsorted/new files (highest priority to classify)

Also scan directory listings (NOT file contents) of:
- Obsidian/Projects/*/Notes/ — to know what project notes already exist
- Clients/*/ — to know what client folders exist

PHASE 2: CLASSIFY
For every file in `notes/`, assign exactly ONE category:

| Category | Meaning | Action |
|----------|---------|--------|
| Access | Credentials, API keys, tokens, passwords, connection strings | Keep in notes/Access/ |
| Archive | Outdated, superseded, no longer relevant | Move to notes/Archive/ |
| Coolify | Coolify/Docker infrastructure | Keep in notes/Coolify/ |
| Posts | Content drafts, blog posts, social media copy | Keep in notes/Posts/ |
| Reference | Commands, how-tos, API docs, dev references | Keep in notes/ root |
| Project:<name> | Relevant to a specific Obsidian/Projects/<name>/ | COPY to project's Notes/ |
| Client:<name> | Relevant to a specific Clients/<name>/ | COPY to client folder |
| Research | Research findings, topic explorations | Move to Research/ |
| Tasks | Actionable to-do items, build lists | Move to tasks/ |
| DELETE | Empty, duplicate, or junk | Delete with justification |

Rules:
- Files already in the correct location → mark as "OK" (no operation needed)
- Files with mixed content (e.g., specs + credentials) → SPLIT into separate files
- Filenames should be kebab-case and descriptive (e.g., `expo-build-commands.md` not `Dev Commands.md`)
- Credentials MUST live in notes/Access/ — flag any found elsewhere as SECURITY
- notes/Review/ is the inbox — every file there needs classification

```

```

For SPLIT operations, also include:

```
FILE: <file path>
ACTION: FLAG
SEVERITY: HIGH | MEDIUM | LOW
ISSUE: <what the concern is>
RECOMMENDATION: <what should be done>
```

Group operations by type: RENAME first, then MOVE, COPY, SPLIT, DELETE, FLAG.
