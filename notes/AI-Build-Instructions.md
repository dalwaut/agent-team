# AI Instructions for Building / Iteration

## Logs
Implement a structured logging system to tell us in the terminal or logs what was loaded, clicked, rendered, fetched / sent data.

## Notes
Create a notes file in the base of the working directory with your understanding of the app and its details.
Include:
- Detailed description of app
- How a problem was fixed

## Tasks
Create a file to be used as a task list for the build. It will have details on what step we are on and the next set of instructions / tasks to do next with details on the current status of this task and how it's going.
