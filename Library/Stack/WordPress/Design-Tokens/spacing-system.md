# Spacing System for Fusion Builder

Standard spacing values for consistent layouts.

## Section Padding (Container Level)

| Context | Top | Bottom | Left/Right |
|---------|-----|--------|------------|
| Hero section | 100-120px | 100-120px | 20px |
| Standard content | 60-80px | 60-80px | 30px |
| CTA banner | 50-60px | 50-60px | 20px |
| Compact section | 30-40px | 30-40px | 20px |
| Decorative divider | 10px | 10px | 0px |
| Full-width map/media | 0px | 0px | 0px |

## Column Padding

| Context | All sides |
|---------|-----------|
| Card with border | 25-30px |
| Content column | 20px |
| Image column | 0px (let image fill) |
| Text beside image | padding-right: 40px (or padding-left if reversed) |

## Column Spacing (Gap)

| Layout | Spacing Value |
|--------|--------------|
| Standard 2-col | 4% |
| Standard 3-col | 4% |
| Tight cards | 2% |
| Wide editorial | 6% |

## Element Spacing

| Between | Margin |
|---------|--------|
| Heading → body text | 15-20px (margin_bottom on title) |
| Body text → button | 25-30px (margin on button) |
| Separator → content | 20-25px (top/bottom margin on separator) |
| Content box cards | 0px (column spacing handles it) |
| Stacked buttons | 15px |

## Scale

Use these values consistently. Pick one and stick with it per project:

**Relaxed**: 20, 40, 60, 80, 100, 120
**Standard**: 15, 30, 60, 80, 100
**Compact**: 10, 20, 40, 60, 80
