## Pricing Design Rules

1. **Highlight one tier** — use dark background, border accent, or shadow to make the recommended option obvious
2. **Align features** — keep feature lists the same length; use "—" for unavailable features
3. **Price prominently** — large font, clear currency, period (per month/year)
4. **CTA on every tier** — never force users to scroll to find the button
5. **3 tiers is optimal** — anchoring effect makes middle tier feel like best value
6. **Outline vs filled buttons** — recommended tier gets filled (primary), others get outline (secondary)
7. **Annual/monthly toggle** — if offering both, consider tabs above the pricing table
