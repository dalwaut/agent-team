## CTA Design Rules

1. **One CTA per section** — don't confuse with multiple actions
2. **Button text = action verb** — "Book Now", "Get Started", "Shop Collection" (not "Click Here")
3. **High contrast** — button must visually pop against background
4. **Concise text** — headline + one sentence + button. No more.
5. **Strategic placement** — after showcasing value (features, testimonials, products)
6. **Final CTA before footer** — every page should end with a call to action
7. **Vary CTA style** — don't use the same pattern twice on one page
