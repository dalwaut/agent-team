# n8n Forge Project

This project was created via the n8n-forge pipeline.

## Structure

| Directory | Purpose |
|-----------|---------|
| `Dev-Plan/plan.md` | Workflow design document (Phase A output) |
| `Codebase/` | Exported n8n workflow JSON + standalone app code |
| `Notes/overview.md` | Project overview and pipeline progress |
| `Research/sources.md` | Research references and existing patterns used |
| `Review-log/review.md` | Forge squad quality gate output |
| `Agent-Tasks/` | Agent task assignments |
| `Ag-Build-Tasks/` | Builder agent work items |
| `Debug-log/` | Debug logs from testing |

## Pipeline

1. Design (plan.md) → 2. Prototype (n8n) → 3. Convert (standalone code) → 4. Review (forge squad) → 5. Ship
