# Web App Template

A basic, vanilla JavaScript/HTML/CSS starter template for web applications.

## Structure

```
Codebase/
├── assets/
│   └── images/
├── css/
│   └── style.css
├── js/
│   └── app.js
└── index.html
```

## Usage

1. Clone or copy this structure.
2. Open `index.html` in your browser.
3. Start editing!
