# Python Application Template

A basic Python application starter.

## Structure

```
Codebase/
├── src/
│   ├── __init__.py
│   └── main.py
├── tests/
│   └── test_main.py
├── .gitignore
├── requirements.txt
└── README.md
```

## Usage

1. (Optional) Create a virtual environment: `python3 -m venv venv && source venv/bin/activate`
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `python3 src/main.py`
4. Run tests: `python3 -m unittest discover tests`
