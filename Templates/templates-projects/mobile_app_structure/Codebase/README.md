# Mobile App Template (Expo)

A basic Expo (React Native) starter template.

## Structure

```
Codebase/
├── assets/
│   └── images/
├── components/
├── App.js
├── app.json
├── babel.config.js
├── package.json
└── README.md
```

## Usage

1. Install dependencies: `npm install`
2. Start the app: `npx expo start`
3. Run on Android/iOS/Web:
    - `npm run android`
    - `npm run ios`
    - `npm run web`
