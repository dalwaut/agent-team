## DESIGN METHODOLOGY

For every page you design, follow this process:

1. **Define the page goal** — What is the single most important action the visitor should take?
2. **Map the content flow** — Arrange sections in order of importance using the inverted pyramid
3. **Apply visual hierarchy** — Size, color, contrast, and spacing guide the eye
4. **Design for mobile first** — Ensure every section stacks gracefully
5. **Apply brand identity** — Colors, typography, and tone from design-specs.md

## VISUAL HIERARCHY PATTERNS

### Home / Landing Page
```
Hero (full-width, max impact)
  → Value Proposition (what + who + why)
  → Features / Offerings (3-column cards or alternating sections)
  → Social Proof (testimonials, logos, gallery)
  → Secondary CTA (newsletter, loyalty, special offer)
  → Featured Products (grid or carousel)
  → Final CTA (contact, book, buy)
  → Footer
```

### About / Story Page
```
Hero Section (brand statement)
  → Origin Story (narrative text, warm tone)
  → Mission & Values (2-column: text + image)
  → What We Offer (content boxes or cards)
  → Who We Serve (audience segments)
  → Promise / Closing Statement
  → CTA (contact or shop)
```

### Services / Portfolio Page
```
Hero Section (title + subtitle)
  → Service Grid (3-4 column cards)
  → Detailed Breakdowns (alternating text + image)
  → Pricing (if applicable)
  → Testimonials
  → CTA (get a quote, book a call)
```

### Location / Events Page
```
Hero Section (title + subtitle)
  → Map (full-width embedded)
  → Upcoming Events (table or card layout)
  → CTA (Book for Private Events)
```

### Contact Page
```
Section 1: Primary Form (booking/inquiry)
  → Decorative Divider
  → Section 2: Secondary Form (general contact)
  → Business Info (address, hours, social links)
```

### Blog / News Page
```
Hero Section (page title)
  → Featured Post (large, full-width)
  → Post Grid (3-column masonry or cards)
  → Pagination or Load More
  → CTA Sidebar or Bottom Banner
```

## BRAND AESTHETIC APPLICATION

Read design-specs.md for the project's specific aesthetic direction. Apply these universal principles:

### Color Pairing Rules
- **Hero sections**: Light background OR full-bleed image with dark overlay + white text
- **Alternating sections**: Alternate between light variant and white for visual rhythm
- **CTA sections**: Primary accent background with contrasting text, or dark background with white text
- **Accent sections**: Secondary color background for special offers/highlights
- **Buttons**: Primary accent background, white text. Hover → slightly darker
- **Borders/dividers**: Use designated accent or border color
- **Never**: Use supporting accents as large backgrounds. Keep them for small elements.

### Typography
- **H1**: Display/serif font, 36-48px, dark text, center-aligned on hero
- **H2**: Display/serif font, 28-36px, dark text
- **H3**: Display/serif font, 22-28px, dark text
- **Body**: Sans-serif, 16-18px, dark text, line-height 1.6-1.8
- **Captions/small**: Sans-serif, 14px, lighter gray
- **Button text**: Sans-serif, 14-16px, uppercase or title case, letter-spacing 1-2px

### Spacing System
- **Between sections (containers)**: 0px margin (use padding instead)
- **Section internal padding**: 60-100px top/bottom, 30px left/right (or 5-8%)
- **Between heading and content**: 20-30px
- **Between content blocks**: 30-40px
- **Column gaps**: 4-6%

## LAYOUT COMPOSITION RULES

### Full-Width Hero
- Background: High-quality image or solid brand color
- Overlay: Dark overlay (rgba black 30-50%) for readability over images
- Content: Centered, max-width 800px
- Elements: H1 title + subtitle paragraph + CTA button
- Minimum height: 60-80vh for home, 40-50vh for inner pages
- Text color: White on image backgrounds, dark on light backgrounds

### Content Sections
- Max content width: 1100-1200px (Avada default)
- Alternating layout: Text-left/Image-right → Image-left/Text-right
- Image columns: 1_2 or 2_5 width
- Text columns: 1_2 or 3_5 width
- Vertical center alignment within flex containers

### Card / Feature Sections
- 3-column layout for offerings (desktop), stacks to 1-column mobile
- Each card: Icon on top → Heading → Short description → Optional link
- Equal height columns with centered content
- Subtle hover effect (liftup or shadow)

### CTA Sections
- Contrasting background color (primary accent, dark, or secondary)
- Centered text: Compelling headline + 1-line supporting text + button
- Button: Large size, pill shape, contrasting color
- Padding: 60-80px vertical

### Decorative Elements
- Section separators: `curved` or `triangle` type between contrasting sections
- Horizontal rules: Single solid, accent color, 1px
- Icons: FontAwesome, used sparingly in content boxes
- Animations: `fade` only, 0.5s speed, triggered on scroll into view

## MOBILE DESIGN RULES

- All multi-column layouts stack to single column on mobile
- Touch targets minimum 44x44px
- Font sizes: H1 reduces to 28-32px, H2 to 22-26px, body stays 16px
- Padding reduces to 30-40px vertical on mobile
- Images scale to 100% width
- Buttons: Full-width on mobile (`stretch="yes"`)
- Map height: Reduce to 300px on mobile
- Hide purely decorative elements on small screens if they clutter
